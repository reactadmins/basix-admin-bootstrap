## Props

<FeatureTable>
    <tr>
        <td>title</td>
        <td>string</td>
        <td></td>
        <td>Show this title card Header.</td>
    </tr>
    <tr>
        <td>titleIcon</td>
        <td>string</td>
        <td></td>
        <td>Set title icon as you like. We recommend class base icons.</td>
    </tr>
    <tr>
        <td>titleBg</td>
        <td>string</td>
        <td></td>
        <td>Set your custom title background with `hex` code</td>
    </tr>
    <tr>
        <td>isHeader</td>
        <td>boolean</td>
        <td>`false`</td>
        <td>If show header then set `true`. By default `false`.</td>
    </tr>
    <tr>
        <td>headerPosition</td>
        <td>`top` | `bottom`</td>
        <td>`top`</td>
        <td>Card header positioned at both the top and bottom of the card</td>
    </tr>
    <tr>
        <td>padding</td>
        <td>string</td>
        <td>`16px`</td>
        <td>Card padding adds spacing between the content and the card edges for better visual appeal and readability.</td>
    </tr>
    <tr>
        <td>bodyBg</td>
        <td>string</td>
        <td></td>
        <td>Card body background defines the color displayed behind the content area of a card, enhancing its visual presentation.</td>
    </tr>
    <tr>
        <td>icons</td>
        <td>array</td>
        <td>`[]`</td>
        <td>icons accept an array of object. ex: <br/> 
        `[ {icon: "fa fa-cog", dropdown: [ { label: "Edit", icon: "fa fa-cog", method: () => alert("Cog"), }, { label: "Delete", icon: "fa-solid fa-trash", method: () => alert("Delete"), }, { label: "Update", icon: "fa-solid fa-recycle", method: () => alert("Update"), }, ], }, { icon: "fa fa-angle-down" }, ]` </td>
    </tr>
    <tr>
        <td>dismissible</td>
        <td>boolean</td>
        <td>`false`</td>
        <td>Card dismissible allows users to easily close or dismiss a card with a click, improving user interaction and flexibility.</td>
    </tr>
    <tr>
        <td>onClose</td>
        <td>func</td>
        <td></td>
        <td>Using onClose with dismissible enables custom actions or functions when a dismissible card is closed, enhancing user experience and functionality.</td>
    </tr>

</FeatureTable>
