# Installation

To use Basix Admin, you need to have `React 18+`, Node.js, npm, yarn and pnpm installed on your machine. Follow these steps to set up the template:

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

#### Clone or download the repository from GitHub

```
www.gtihub.com
```

#### Run to install the dependencies

<Tabs>
    <TabItem value="npm" label="npm">
        ``` jsx npm install ```
    </TabItem>
    <TabItem value="yarn" label="yarn">
        ``` jsx yarn install ```
    </TabItem>
    <TabItem value="pnpm" label="pnpm">
        ``` jsx pnpm install ```
    </TabItem>
</Tabs>

#### Run to start the development server

<Tabs>
    <TabItem value="npm" label="npm">
        ``` jsx npm run dev ```
    </TabItem>
    <TabItem value="yarn" label="yarn">
        ``` jsx yarn run dev ```
    </TabItem>
    <TabItem value="pnpm" label="pnpm">
        ``` jsx pnpm run dev ```
    </TabItem>
</Tabs>

#### Run to build the project

<Tabs>
    <TabItem value="npm" label="npm">
        ``` jsx npm run build ```
    </TabItem>
    <TabItem value="yarn" label="yarn">
        ``` jsx yarn run build ```
    </TabItem>
    <TabItem value="pnpm" label="pnpm">
        ``` jsx pnpm run build ```
    </TabItem>
</Tabs>

#### Run to preview the project

<Tabs>
    <TabItem value="npm" label="npm">
        ``` jsx npm run preview ```
    </TabItem>
    <TabItem value="yarn" label="yarn">
        ``` jsx yarn run preview ```
    </TabItem>
    <TabItem value="pnpm" label="pnpm">
        ``` jsx pnpm run preview ```
    </TabItem>
</Tabs>

Open your browser and go to http://localhost:5173/
You should see the Basix Admin Template on your screen. You can explore the different pages and components, and customize them according to your needs. To build the production version of the template, run `npm, yarn & pnpm run build`. This will create a `build` folder with the optimized files. To serve the production version, run `npm, yarn & pnpm run preview` and go to http://localhost:4173/ again.
